# ML-DL-samples
Project :
Bike Rental Prediction using Linear Regression and Random Forest

Overview :
This project predicts the total count of bike rentals (cnt) using machine learning algorithms. Two approaches are implemented:

Linear Regression
Random Forest Regression

Both models are trained and evaluated on the same dataset (bike.csv) to compare performance.

Repository Contents:

linear_regression.py → Code for Linear Regression model

random_forest.py → Code for Random Forest model

bike.csv → Dataset containing bike rental information

README.md → Documentation

📊 Dataset :
The dataset (bike.csv) includes features such as:

Weather conditions
Season
Temperature
Humidity
Wind speed
Holiday/working day indicators

Target variable:

cnt → Total count of bike rentals
